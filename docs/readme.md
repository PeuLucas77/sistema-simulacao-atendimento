Forma B: Fazendo pelo seu computador (O jeito mais profissional)
Se você já baixou o Git no seu computador, essa forma é muito mais parecida com o que você fará no dia a dia como Cientista de Dados.

Abra uma pasta vazia no seu computador (pode ser na Área de Trabalho).

Clique com o botão direito dentro dela e abra o Git Bash (ou abra o prompt de comando/terminal e navegue até ela).

Clone o seu repositório do GitHub para o seu computador digitando:

Bash
git clone https://github.com/seu-usuario/nome-do-seu-repositorio.git
Entre na pasta que foi criada. Você verá que o arquivo README.md que o GitHub criou já está lá.

Crie as pastas normalmente pelo Windows/Linux/Mac:

Crie uma pasta chamada src e jogue o seu arquivo atendimento.py dentro dela.

Crie uma pasta chamada docs e jogue os arquivos relatorio.pdf e testes.pdf dentro dela.
